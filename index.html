Initiation à la Carte Micro:bit

Présentation Générale

La carte Micro:bit est un microcontrôleur éducatif développé par la BBC pour initier les jeunes à la programmation et à l'électronique. Elle intègre de nombreux capteurs qui permettent de réaliser des projets créatifs et éducatifs.

Face Avant

  • Écran LED 5x5
  • Boutons A et B
  • Connecteur d'extension
  • LED d'alimentation

Face Arrière

  • Processeur ARM Cortex-M0
  • Accéléromètre et boussole
  • Capteur de température
  • Connecteur USB

Caractéristiques Techniques

Processeur

ARM Cortex-M0

Mémoire

16KB RAM

Capteurs

5 intégrés

Connectivité

Bluetooth

Premiers Pas

1

Télécharger l'éditeur

Rendez-vous sur le site officiel Micro:bit pour télécharger l'éditeur de programmation.

2

Connecter la carte

Utilisez un câble USB pour connecter la Micro:bit à votre ordinateur.

3

Programmer

Créez votre premier programme en blocs ou en Python.

4

Téléverser

Téléversez le programme sur la carte et testez-le immédiatement.

TP 1: L'Accéléromètre

Objectifs du TP

  • Comprendre le fonctionnement d'un accéléromètre
  • Mesurer les accélérations sur 3 axes (X, Y, Z)
  • Créer un niveau à bulle numérique
  • Détecter les mouvements et les chutes

Théorie

L'accéléromètre mesure l'accélération linéaire selon trois axes. Sur la Micro:bit :

  • X : Gauche/Droite
  • Y : Avant/Arrière
  • Z : Haut/Bas

Matériel Nécessaire

  • Carte Micro:bit
  • Câble USB
  • Ordinateur avec éditeur
  • Support inclinable (optionnel)

Exemple Pratique

Exercice 1: Afficher les valeurs

Solution en Blocs

Solution en Python

from microbit import *

while True:
    # Afficher l'accélération sur l'axe X
    display.scroll(str(accelerometer.get_x()))
    sleep(100)
    display.clear()
    
    # Afficher l'accélération sur l'axe Y
    display.scroll(str(accelerometer.get_y()))
    sleep(100)
    display.clear()
    
    # Afficher l'accélération sur l'axe Z
    display.scroll(str(accelerometer.get_z()))
    sleep(100)
    display.clear()

Exercice 2: Détecteur de chute

Énoncé

Créez un programme qui affiche un visage triste lorsque la Micro:bit détecte une chute (accélération brutale sur l'axe Z).

Indice : Utilisez l'événement "est secoué"

Correction Python

from microbit import *

while True:
    if accelerometer.is_gesture("shake"):
        display.show(Image.SAD)
        sleep(2000)
    else:
        display.show(Image.HAPPY)
    sleep(100)

TP 2: La Boussole

Objectifs du TP

  • Comprendre le fonctionnement d'une boussole numérique
  • Calibrer la boussole de la Micro:bit
  • Afficher les points cardinaux
  • Créer un indicateur de direction

Théorie

La boussole de la Micro:bit utilise le champ magnétique terrestre pour déterminer l'orientation. Elle mesure les degrés par rapport au nord magnétique (0° = Nord).

Points Cardinaux

  • Nord : 0°
  • Est : 90°
  • Sud : 180°
  • Ouest : 270°

Calibration

Important !

La boussole doit être calibrée avant la première utilisation. Suivez les instructions à l'écran : faites tourner la Micro:bit pour dessiner un cercle avec les LEDs.

Éloignez-vous des objets métalliques pendant la calibration

Exercice 1: Afficher la direction

Solution en Blocs


                            

                        

                        
                        

                            
Solution en Python

                            

                                
from microbit import *

# Calibrer la boussole
compass.calibrate()

while True:
    heading = compass.heading()
    
    if heading < 45:
        display.show("N")
    elif heading < 135:
        display.show("E")
    elif heading < 225:
        display.show("S")
    elif heading < 315:
        display.show("O")
    else:
        display.show("N")
    
    sleep(500)

                            

                        

                    

                    
                    
Exercice 2: Boussole numérique

                    

                        

                            
Énoncé

                            

                                Créez une boussole numérique qui affiche une flèche pointant vers le nord. 
                                La flèche doit tourner en fonction de l'orientation de la Micro:bit.
                            

                            

                                
                                Indice : Utilisez les images prédéfinies Image.ARROW_N, Image.ARROW_E, etc.
                            

                        

                        
                        

                            
Correction Python

                            

                                
from microbit import *

compass.calibrate()

while True:
    heading = compass.heading()
    
    # Déterminer la direction la plus proche
    if heading < 45 or heading >= 315:
        display.show(Image.ARROW_N)
    elif heading < 135:
        display.show(Image.ARROW_E)
    elif heading < 225:
        display.show(Image.ARROW_S)
    else:
        display.show(Image.ARROW_W)
    
    sleep(100)

                            

                        

                    

                

            

        


        
        

            

                
TP 3: Le Capteur de Température

                
                

                    
Objectifs du TP

                    
    
                        
  • Mesurer la température ambiante
    • 
                        
  • Comprendre la différence température processeur/ambiante
    • 
                        
  • Créer un thermomètre numérique
    • 
                        
  • Implémenter des alertes de température
    • 
                    

                    
                    

                        

                            
Théorie

                            

                                La Micro:bit possède deux sources de mesure de température :
                            

                            

                                

                                    
Température du processeur

                                    
Mesure la chaleur générée par le microcontrôleur lui-même.

                                

                                

                                    
Température ambiante

                                    
Mesure approximative de la température de l'environnement.

                                

                            

                            

                                
                                Pour une mesure précise de la température ambiante, attendez que la carte soit éteinte depuis plusieurs minutes.
                            

                        

                        
                        

                            
Applications pratiques

                            

                                

                                    
                                    

                                        
Thermomètre domestique

                                        
Surveiller la température d'une pièce

                                    

                                

                                

                                    
                                    

                                        
Alerte surchauffe

                                        
Détecter la surchauffe d'un équipement

                                    

                                

                                

                                    
                                    

                                        
Station météo

                                        
Enregistrer les variations de température

                                    

                                

                            

                            
                            
                        

                    

                    
                    
Exercice 1: Thermomètre basique

                    

                        

                            
Solution en Blocs

                            

                                

                            

                        

                        
                        

                            
Solution en Python

                            

                                
from microbit import *

while True:
    temp = temperature()
    display.scroll(str(temp) + "C")
    sleep(2000)
    display.clear()
    sleep(1000)

                            

                        

                    

                    
                    
Exercice 2: Alerte de température

                    

                        

                            
Énoncé

                            

                                Créez un programme qui affiche un visage heureux lorsque la température est normale (18-25°C), 
                                un visage neutre si elle est acceptable (10-18°C ou 25-30°C), et un visage triste si elle est 
                                critique (<10°C ou >30°C).
                            

                            

                                
                                Indice : Utilisez les images Image.HAPPY, Image.MEH, Image.SAD
                            

                        

                        
                        

                            
Correction Python

                            

                                
from microbit import *

while True:
    temp = temperature()
    
    if temp >= 18 and temp <= 25:
        display.show(Image.HAPPY)
    elif (temp >= 10 and temp < 18) or (temp > 25 and temp <= 30):
        display.show(Image.MEH)
    else:
        display.show(Image.SAD)
    
    sleep(5000)  # Vérifier toutes les 5 secondes

                            

                        

                    

                

            

        


        
        

            

                
TP 4: Le Capteur de Lumière

                
                

                    
Objectifs du TP

                    
    
                        
  • Mesurer l'intensité lumineuse ambiante
    • 
                        
  • Utiliser l'écran LED comme capteur de lumière
    • 
                        
  • Créer un détecteur de luminosité
    • 
                        
  • Implémenter un système d'éclairage automatique
    • 
                    

                    
                    

                        

                            
Fonctionnement

                            

                                La Micro:bit utilise son écran LED 5x5 comme capteur de lumière. 
                                Chaque LED peut détecter la quantité de lumière qui l'atteint.
                            

                            

                                
Comment ça marche ?

                                

                                    Les LEDs sont brièvement éteintes et mesurent la lumière ambiante. 
                                    La valeur retournée est une moyenne des 25 capteurs.
                                

                            

                            
    
                                
  • 0 : Obscurité totale
    • 
                                
  • 255 : Lumière maximale
    • 
                                
  • ~50 : Pièce normalement éclairée
    • 
                                
  • ~150 : Lumière du jour indirecte
    • 
                            

                        

                        
                        

                            
Applications

                            

                                

                                    

                                        
                                    

                                    

                                        
Détecteur de nuit/jour

                                        
Allumer/éteindre des LEDs automatiquement

                                    

                                

                                

                                    

                                        
                                    

                                    

                                        
Jardinage intelligent

                                        
Surveiller l'ensoleillement des plantes

                                    

                                

                                

                                    

                                        
                                    

                                    

                                        
Domotique

                                        
Automatiser l'éclairage d'une pièce

                                    

                                

                            

                        

                    

                    
                    
Exercice 1: Mesure de luminosité

                    

                        

                            
Solution en Blocs

Exercice 2: Détecteur de nuit

                    

                        

                            
Énoncé

                            

                                Créez un programme qui affiche une lune lorsque la luminosité est faible (nuit) 
                                et un soleil lorsque la luminosité est élevée (jour). Utilisez le seuil de 50.
                            

                            

                                
                                Indice : Créez vos propres images avec Image()
                            

                        

                        
                        

                            
Correction Python

                            

                                
from microbit import *

# Créer des images personnalisées
sun = Image("09090:"
            "99999:"
            "09990:"
            "99999:"
            "09090")

moon = Image("00900:"
             "09090:"
             "90009:"
             "09090:"
             "00900")

while True:
    light = display.read_light_level()
    
    if light < 50:
        display.show(moon)
    else:
        display.show(sun)
    
    sleep(5000)

                            

                        

                    

                

            

        


        
        

            

                
TP 5: Les Boutons A et B

                
                

                    
Objectifs du TP

                    
    
                        
  • Gérer les événements des boutons A et B
    • 
                        
  • Détecter les appuis courts et longs
    • 
                        
  • Créer des interfaces interactives
    • 
                        
  • Implémenter un compteur et un sélecteur
    • 
                    

                    
                    

                        

                            
Les boutons de la Micro:bit

                            

                                

                                    

                                        A
                                    

                                    

                                        B
                                    

                                    

                                        

                                    

                                

                            

                            

                                La Micro:bit possède trois boutons physiques : A, B et le bouton reset. 
                                Les boutons A et B sont programmables et peuvent déclencher des actions.
                            

                        

                        
                        

                            
Types d'événements

                            

                                

                                    
button_a.is_pressed()

                                    
Vérifie si le bouton est actuellement pressé

                                

                                

                                    
button_a.was_pressed()

                                    
Vérifie si le bouton a été pressé depuis la dernière vérification

                                

                                

                                    
button_a.get_presses()

                                    
Compte le nombre d'appuis depuis le dernier appel

                                

                            

                        

                    

                    
                    
Exercice 1: Compteur avec boutons

                    

                        

                            
Solution en Blocs

                            

                                

                            

                        

                        
                        

                            
Solution en Python

                            

                                
from microbit import *

counter = 0
display.show(counter)

while True:
    if button_a.is_pressed():
        counter += 1
        display.show(counter)
        sleep(200)
    
    if button_b.is_pressed():
        counter -= 1
        display.show(counter)
        sleep(200)

                            

                        

                    

                    
                    
Exercice 2: Jeu de réaction

                    

                        

                            
Énoncé

                            

                                Créez un jeu de réaction : après un délai aléatoire, affichez une flèche. 
                                Le joueur doit appuyer sur le bouton correspondant (A pour gauche, B pour droite) 
                                le plus rapidement possible.
                            

                            

                                
                                Indice : Utilisez random.randint() pour le délai aléatoire
                            

                        

                        
                        

                            
Correction Python

                            

                                
from microbit import *
import random

while True:
    # Attente aléatoire
    wait_time = random.randint(1000, 5000)
    sleep(wait_time)
    
    # Choisir une direction aléatoire
    direction = random.choice(["left", "right"])
    
    if direction == "left":
        display.show(Image.ARROW_W)
        start_time = running_time()
        
        while True:
            if button_a.is_pressed():
                reaction_time = running_time() - start_time
                display.scroll(str(reaction_time) + "ms")
                break
            elif button_b.is_pressed():
                display.show(Image.NO)
                sleep(1000)
                break
    else:
        display.show(Image.ARROW_E)
        start_time = running_time()
        
        while True:
            if button_b.is_pressed():
                reaction_time = running_time() - start_time
                display.scroll(str(reaction_time) + "ms")
                break
            elif button_a.is_pressed():
                display.show(Image.NO)
                sleep(1000)
                break
    
    sleep(2000)

Station Météo Intelligente

                        

                            Utilisez toutes les connaissances acquises pour créer une station météo complète avec la Micro:bit
                        

                    

                    
                    

                        

                            
Énoncé du Projet

                            

                                

                                    
Objectif Principal

                                    

                                        Créer une station météo portable qui mesure et affiche en temps réel :
                                        température, luminosité, orientation et mouvement.
                                    

                                

                                
                                

                                    
Fonctionnalités Requises

                                    
    
                                        
  • Affichage cyclique des mesures
    • 
                                        
  • Sélection manuelle des données via boutons
    • 
                                        
  • Alertes visuelles pour valeurs critiques
    • 
                                        
  • Enregistrement des données minimal/maximal
    • 
                                        
  • Interface utilisateur intuitive
    • 
                                    

                                

                                
                                

                                    
Contraintes Techniques

                                    
    
                                        
  • Utiliser uniquement les capteurs intégrés
    • 
                                        
  • Programme en Python ou blocs
    • 
                                        
  • Batterie incluse pour portabilité
    • 
                                        
  • Documentation du code requise
    • 
                                    

                                

                            

                        

                        
                        

                            
Exemples d'Implémentation

                            

                                

                                    
Interface Utilisateur

                                    
                                    

                                        Utilisez les boutons A/B pour naviguer entre les différentes mesures. 
                                        L'écran affiche l'icône correspondante et la valeur.
                                    

                                

                                
                                

                                    
Structure Recommandée

   - Affichage formaté
   - Gestion des records
   - Alertes visuelles
